Sistema-de-Gerenciamento-de-Alunos-em-Python
Sistema simples de cadastro de alunos desenvolvido em Python utilizando programação orientada a objetos. O programa permite registrar informações de alunos (nome, idade, nota e faltas) e verificar automaticamente sua situação acadêmica.
Funcionalidades:

Cadastro de alunos com nome, idade, nota e faltas
Verificação automática de aprovação por nota (média 6.0)
Sistema de recuperação que adiciona 2 pontos à nota
Verificação de reprovação por faltas (limite de 100 faltas)
Feedback sobre a situação final do aluno

Tecnologias:

Python 3

Como usar:
Execute o arquivo principal e acompanhe as verificações de aprovação do aluno cadastrado.
