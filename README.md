# T-tulo-Sistema-de-Gerenciamento-de-Alunos-em-Python
Sistema de Gerenciamento de Alunos Sistema simples em Python para cadastro de alunos e verificação de aprovação. Utiliza POO para gerenciar nome, idade, notas e faltas. Inclui sistema de recuperação e validação automática de aprovação/reprovação. Tecnologias: Python 3
