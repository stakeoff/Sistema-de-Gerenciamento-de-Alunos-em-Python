Sistema de Alunos em Python
Um sistema simples de cadastro de alunos feito em Python 3 – ideal pra quem tá aprendendo POO!
Funcionalidades

Cadastro de alunos (nome, idade, nota, faltas)
Sistema de recuperação (+2 pontos na nota)
Verificação automática de aprovação/reprovação
Controle de faltas (limite: 100 faltas)

Como usar

Baixe ou clone o repositório
Abra o terminal na pasta do projeto
Rode o comando:

python aluno.py
